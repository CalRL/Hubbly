# Hubbly

> A modern, modular, and extensible action-based plugin for Minecraft servers.

Hubbly makes it easy to manage and enhance your server without the hassle. With a powerful action system and simple configuration, you can create everything from small quality-of-life features to fully custom experiences, all without needing complex setups. Whether you're running a small server or a large network, Hubbly keeps things flexible, reliable, and easy to control.

## 🚀 Features

- **Actions System** *(See Wiki)* — chain actions (messages, commands, items, etc)
- **Compass Server Selector**, **Launchpads**, **Shop Item**
- **BossBars (animated)**, **Auto Announcements**, **Socials Item**
- **Player Visibility Toggle**, **Anti Void Fall**
- **Spawn & SetSpawn**, **Custom Items**
- **Disable Worlds**, **Anti World Download**
- Enderbow, Grappling Hook, and other mobility items

### 👋 Join & Player Events
- Custom join events
    - Items
    - Messages
    - Title
    - Other Actions
- Custom join/leave messages

### 🌍 World Event Management
- Control/cancel: block place/break, damage, weather, item drop/pickup, hunger, mob spawning, block burn, leaf decay, projectiles


## 📦 Installation

1. Download the latest release `.jar`
2. Place it into your server's `plugins/` folder
3. Start your Paper / Spigot server
4. Stop the server after first run (to generate config files)
5. Configure:
   - `config.yml`
   - language files (`en.yml`, etc.)
6. Restart the server
